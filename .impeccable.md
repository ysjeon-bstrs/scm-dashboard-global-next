# Impeccable Design Context

This file persists the design context for the SCM Dashboard Global Next project so
every future design session starts from the same intent. Keep it in sync when the
product direction changes.

## Design Context

### Users
Internal Boosters supply-chain operators (boosters.kr Google accounts). They work in
this tool daily to read inventory snapshots, track inbound/outbound logistics, and run
FEFO lot allocation against company stock. Sessions are task-focused: scan a lot table,
upload a request file, confirm allocation, export. The interface is read-heavy and
number-heavy — accuracy and fast scanning matter more than decoration.

### Brand Personality
Trustworthy and precise — three words: **calm, exact, dependable**. The tool should
feel like a well-made instrument, not a marketing site. Confidence comes from legible
numbers, clear status, and quiet surfaces — never from flashy effects. It should read as
"someone who cares about correctness built this."

### Aesthetic Direction
Refined operational instrument. Light theme only. A tinted near-white paper surface,
deep blue-teal ink text, and a single deep teal-green brand accent used sparingly for
primary actions and emphasis. Status is communicated through a disciplined three-state
color system (ok / warning / shortage). Typography is Pretendard (KO + EN) with a strong
size/weight hierarchy and tabular numerals for all quantities; monospace is reserved for
real codes (SKU, lot numbers) where it aids scanning.

- **Anti-references**: generic slate-on-white admin templates, evenly-distributed
  same-size card grids, Inter/system-default type, purple-blue gradients, glassmorphism,
  dark mode with glowing accents.

### Design Principles
1. **Numbers lead.** Quantities are the content — give them tabular numerals, generous
   size, and the strongest weight. Labels stay quiet (small, tinted, uppercase tracking).
2. **Status is color; everything else is neutral.** Color carries meaning (brand =
   action, ok/warn/danger = allocation state). Neutral surfaces never compete with it.
3. **Tinted neutrals, never pure black/white.** All neutrals and the accent share a
   teal-leaning hue (OKLCH) so the palette feels cohesive and calm.
4. **Hierarchy through rhythm, not boxes.** Vary spacing and weight to group meaning;
   avoid nesting cards in cards and repeating identical containers.
5. **One quiet, orchestrated entrance.** A single subtle staggered page-load reveal,
   reduced-motion safe — no scattered micro-animations.
