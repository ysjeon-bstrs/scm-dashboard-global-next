## 회고(retrospective) 운영

프로젝트 고유의 컨벤션·결정·함정은 `retrospective.md`에 기록돼 있다(아래 import로 매 세션 로드됨). 관련 작업을 시작하기 전 해당 섹션을 먼저 확인한다.

의미 있는 작업 단위를 마쳤을 때(기능 완료, 버그 수정 완료, 커밋/PR 직전 등) **먼저 `/wrapup` 실행을 사용자에게 능동적으로 제안한다.** 사용자가 끝났다고 말하기를 기다리지 말 것. 단, 승인 없이 retrospective.md를 직접 수정하지는 않는다(제안 → 승인 후 반영).
